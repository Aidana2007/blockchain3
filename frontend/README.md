# Frontend Structure

The frontend is organized into multiple focused pages with navigation:

## Pages

- **index.html** - Home page with login, wallet connection, and quick links
- **skins.html** - Browse and purchase skins
- **campaigns.html** - View and fund crowdfunding campaigns
- **create-campaign.html** - Create new campaigns
- **buy-stm.html** - Purchase STM tokens with ETH
- **wallet.html** - Manage balance, transfers, and transaction history

## Navigation

Each page includes a consistent navigation bar at the top for easy access to all features.

## Scripts

- **app.js** - Main application logic
- **auth.js** - Authentication service
- **blockchain.js** - Blockchain interaction
- **market.js** - Market data
- **ui.js** - UI utilities
